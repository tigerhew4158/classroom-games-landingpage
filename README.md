
# 课堂游戏模板网站 Landing Page（三语商业销售强化版 v17）

本版重点：Supabase 后台订单数据库版。

## 已新增功能

- 订单写入 Supabase 数据库
- 付款截图上传到 Supabase Storage
- 新增管理员后台：`admin.html`
- 管理员可查看：
  - 购买者姓名
  - 电话
  - 电邮
  - 购买配套
  - 模板编号
  - 原价
  - 特价
  - 优惠金额
  - 付款截图
  - 订单状态
  - 管理员备注
- 管理员可更新订单状态：
  - 待确认
  - 已确认付款
  - 已发送账号
  - 取消 / 有问题

## 保留功能

- CN / EN / BM 三语切换
- 所有按钮可用
- DuitNow / Touch n Go QR 付款图
- 上传付款截图
- WhatsApp 手动发送给阿虎老师
- RM400 全站模板
- 原价 RM730，节省 45%
- 所有游戏画面与宣传图

## 需要设置

`README_BACKEND_SETUP.md`

部署后打开：

`/admin.html`
